# Binary-MIPS32
编译了在验证漏洞以及挖掘漏洞时所需要的一些工具,可能设备本身没有。

gdbserver: ELF 32-bit LSB executable, MIPS, MIPS32 rel2 version 1 (SYSV), statically linked, stripped 

atftp: ELF 32-bit LSB executable, MIPS, MIPS32 version 1 (SYSV), statically linked, stripped 

busybox: ELF 32-bit LSB executable, MIPS, MIPS32 version 1 (SYSV), statically linked, stripped 

socat1: ELF 32-bit LSB executable, MIPS, MIPS32 version 1 (SYSV), statically linked, stripped 

strace: ELF 32-bit LSB executable, MIPS, MIPS32 version 1 (SYSV), statically linked, stripped 

tcpdump: ELF 32-bit LSB executable, MIPS, MIPS32 version 1 (SYSV), statically linked, stripped
